# Cas_armados

Material publico de formacion CAS para Arma Reforger, centrado en apoyo aereo cercano con helicopteros dentro de un entorno de simulacion y juego cooperativo.

El objetivo del repositorio es reunir documentacion clara, practica y navegable para que alumnos, pilotos, JTAC e instructores puedan preparar sesiones, consultar procedimientos y mantener un criterio comun durante la instruccion.

## Sitio publicado

La version navegable esta disponible en GitHub Pages:

- Manual del alumno: <https://dmp-spain.github.io/Cas_armados/>
- Chuletas CAS: <https://dmp-spain.github.io/Cas_armados/CHULETAS_CAS.html>
- Manual de instructores: <https://dmp-spain.github.io/Cas_armados/MANUAL_INSTRUCTORES_CAS.html>
- Guia Dia 1 - Operacion FORD: <https://dmp-spain.github.io/Cas_armados/guia-dia1-operacion-ford.html>

## Que contiene

### Manual del alumno

Documento principal para entender el ciclo completo de una mision CAS con helicopteros: briefing, comunicaciones, salida, navegacion, contacto con JTAC, secuencia de ataque, restricciones, regreso y debrief.

Esta pensado para formar una base comun antes de entrar en procedimientos mas exigentes.

### Chuletas CAS

Tarjetas de consulta rapida para usar durante preparacion, instruccion o partida. Incluyen vocabulario de radio, 9-Line, CCA 5-Line, reglas de seguridad, patrones de espera, contingencias y herramientas rellenables.

Su funcion no es sustituir al manual, sino reducir la carga mental cuando ya se conoce el procedimiento.

### Manual de instructores

Guia para preparar y dirigir sesiones de formacion. Ordena la progresion por fases, propone ejercicios, criterios de evaluacion, errores frecuentes y una forma comun de conducir la instruccion sin improvisar cada clase desde cero.

### Guia Dia 1 - Operacion FORD

Sesion inicial estructurada para introducir comunicaciones, gestion de grupo, circuito, contacto JTAC basico y contingencias sencillas. Esta orientada a que el primer dia sea practico, controlado y comprensible.

## Enfoque del proyecto

Este material no pretende ser doctrina militar real ni documentacion oficial. Es una adaptacion para Arma Reforger, con prioridad en:

- claridad operativa;
- seguridad de la partida;
- comunicaciones breves y utiles;
- procedimientos repetibles;
- aprendizaje progresivo;
- coordinacion entre tierra y aire.

Cuando se simplifica un concepto, se hace para que funcione dentro del contexto del servidor y de la formacion. La prioridad es que el grupo entienda que debe hacer, cuando hacerlo y como comunicarlo sin saturar la radio.

## Publico previsto

El repositorio esta pensado para:

- alumnos que empiezan en CAS;
- pilotos de helicoptero que necesitan una secuencia comun;
- JTAC o lideres en tierra que coordinan apoyo aereo;
- instructores que preparan sesiones de formacion;
- miembros de la comunidad que quieran consultar el material publicado.

## Formato

Los documentos vigentes se publican en HTML para poder abrirlos directamente desde el navegador y consultarlos sin herramientas adicionales.

No hay una version PDF vigente en este repositorio. Si circula una copia en PDF, puede estar desactualizada frente a la version HTML publicada.

## Estado

El contenido esta vivo y puede cambiar conforme evolucione la formacion, cambien las necesidades del grupo o se detecten partes mejorables durante las sesiones.

La intencion es mantener el repositorio como referencia estable, pero no congelada: los procedimientos deben poder corregirse cuando la practica demuestre que algo se explica mal, sobra o falta.

## Contribuciones

Las correcciones utiles son bienvenidas, especialmente si mejoran claridad, consistencia o aplicacion practica dentro de Arma Reforger.

Antes de proponer cambios conviene revisar el material publicado para evitar duplicar secciones o introducir procedimientos que contradigan la progresion existente.
